# professional-residency-management-system
Web-based system for managing professional residency processes using PHP and MySQL.
Web-based system designed to manage professional residency processes for students.

## Technologies
- PHP
- MySQL
- HTML / CSS / JavaScript

## Features
- Student registration
- Residency process management
- Database-driven system

## Setup
Update the database credentials in the configuration file using your local database.

## Security
Database credentials have been removed from this repository.
